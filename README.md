# 🍎 Toko Buah Lili - Landing Page

Landing page modern dan responsif untuk **Toko Buah Segar Lili** yang berlokasi di Waluya, Cikarang Utara. Didesain untuk memaksimalkan konversi pelanggan melalui integrasi WhatsApp yang cerdas dan visual produk yang menggugah selera.

## 🚀 Fitur Utama

- **Modern Bento Grid**: Menampilkan koleksi produk (buah, sayur, parsel) dengan desain grid yang interaktif.
- **Smart WhatsApp Integration**: Tombol chat otomatis menyesuaikan pesan berdasarkan produk yang dipilih dan mendeteksi jam operasional toko.
- **Visual Premium**: Animasi halus menggunakan Framer Motion dan tipografi modern dengan font *Outfit*.
- **SEO Optimized**: Meta tags lengkap untuk meningkatkan visibilitas di area Cikarang Utara dan Waluya.
- **Google Maps Interaction**: Lokasi toko yang tersemat secara interaktif di bagian footer.

## 🛠️ Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Font**: [Outfit](https://fonts.google.com/specimen/Outfit) via Google Fonts

## 🏃 Cara Menjalankan Project

1. **Clone repository & Install dependencies**:
   ```bash
   npm install
   ```

2. **Jalankan server pengembangan**:
   ```bash
   npm run dev
   ```

3. **Buka browser**:
   Akses [http://localhost:3000](http://localhost:3000) untuk melihat hasilnya.

## 📁 Struktur Direktori Penting

- `src/app`: Konfigurasi Router dan Layout Utama.
- `src/components`: Berisi komponen UI mandiri (Hero, BentoGrid, Navbar, dll).
- `src/components/WhatsAppCTA.tsx`: Logika deteksi waktu operasional dan pemesanan.

## 📝 Catatan Tambahan

Project ini menggunakan konfigurasi `next.config.ts` untuk mengizinkan gambar dari `images.unsplash.com`. Semua konten visual saat ini merupakan placeholder untuk kebutuhan demonstrasi.
